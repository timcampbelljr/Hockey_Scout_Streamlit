# Hockey_Scout_Streamlit
Player Cards
Each Player on the Crunch will have their own Player Card with the option to select eveyplay on the team. Each Skater will have their season long box score, shot charts, xG Per Shot, Shootout Data when possible, and faceoffs. For Goalies, Their will be box scores which include Goals Against Average, Save%, Minutes/Game Played. Also Thier will be Shot Charts show where they let goals up on the ice.
